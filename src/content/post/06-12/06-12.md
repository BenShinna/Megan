### 第一部分：Highlighted Parts

#### 1. informed decisions
*   **单词**：**informed** [ɪnˈfɔːmd] (adj.)
    *   **释义**：见多识广的；基于充分信息的；明智的。
    *   **词根**：inform (v. 通知) + -ed。
*   **短语**：**make informed decisions**
    *   **释义**：做出明智的决定；在充分了解情况的基础上做决定。
    *   **原文语境**：...helping consumers make **informed decisions** about food... (帮助消费者对食物做出明智的决定)。

#### 2. ultimately
*   **单词**：**ultimately** [ˈʌltɪmətli] (adv.)
    *   **释义**：最终；最后；根本地。
    *   **同义词**：finally, eventually, in the end.
    *   **原文语境**：...and **ultimately** prevent unsafe consumption... (并最终防止不安全的食用...)。这里表示最终的目的或结果。

#### 3. 2-date labeling system
*   **单词**：**labeling** (英式拼写常为 labelling) [ˈleɪbəlɪŋ] (n.)
    *   **释义**：贴标签；标签系统。
*   **短语**：**the 2-date labeling system**
    *   **释义**：双日期标签系统。
    *   **背景**：指美国常见的两种标签："BEST If Used By"（最佳食用日期，关乎质量）和 "USE By"（使用截止日期，关乎安全）。

#### 4. educational messaging that
*   **单词**：**messaging** [ˈmesɪdʒɪŋ] (n.)
    *   **释义**：（通过广告、宣传等传递的）信息；消息传递。
    *   **用法**：这里指教育性的宣传信息。
*   **语法**：**that explained the labels' meaning** 是定语从句，修饰 messaging。
    *   **原文语境**：...reading **educational messaging that** explained the labels' meaning. (阅读解释标签含义的教育信息)。

#### 5. respondents
*   **单词**：**respondent** [rɪˈspɒndənt] (n.) -> 复数 **respondents** [rɪˈspɒndənts]
    *   **释义**：（调查或问卷的）受访者；回答者。
    *   **词根**：respond (v. 回答) + -ent (人)。
    *   **原文语境**：Less than half (46 percent) of study **respondents** knew... (不到一半的研究受访者知道...)。

#### 6. framing the messages with values
*   **单词**：**framing** [freɪmɪŋ] (v. frame的现在分词)
    *   **释义**：构建；表达；制定（框架）。在传播学中，"Framing" 指通过特定的角度或价值观来呈现信息，以影响受众的理解。
    *   **单词**：**values** [ˈvæljuːz] (n.)
    *   **释义**：价值观；重要性。
*   **短语解析**：**framing the messages with values**
    *   **释义**：用价值观来包装/构建信息。
    *   **原文语境**：Researchers explored if **framing the messages with values**, like saving money or avoiding waste... (研究者探索如果用价值观——如省钱或避免浪费——来包装信息，是否会影响效果)。

#### 7. another
*   **单词**：**another** [əˈnʌðə(r)] (pron.)
    *   **释义**：另一个。
    *   **原文语境**：None of the seven value frames tested was significantly more effective at increasing understanding than **another**...
    *   **解析**：这里指测试的7种价值观框架中，没有哪一种比**另一种**（any other one）显著更有效。意思是这7种方法效果差不多。

#### 8. less attentive
*   **单词**：**attentive** [əˈtentɪv] (adj.)
    *   **释义**：注意的；留心的；关心的。
    *   **搭配**：pay attention to -> be attentive to.
    *   **原文语境**：...may make consumers **less attentive** to educational messaging... (可能导致消费者对教育信息**不那么留心/关注**)。
    *   **逻辑**：因为消费者觉得自己懂了（misplaced confidence），所以就不认真看解释了。

```astro
import slateConfig from '~@/slate.config';

interface Props {
  frontSrc?: string;
  backSrc?: string;
  alt?: string;
  size?: number;
}

const {
  frontSrc = slateConfig.avatar!,
  backSrc = slateConfig.avatarBack || slateConfig.avatar!,
  alt = slateConfig.title,
  size = 56,
} = Astro.props;

const sizePx = `${size}px`;
---

<div class="avatar-flip-container" style={`width: ${sizePx}; height: ${sizePx};`}>
  <div class="avatar-flipper">
    <div class="avatar-front">
      <img
        src={frontSrc}
        alt={alt}
        class="avatar-img"
        width={size}
        height={size}
        loading="eager"
      />
    </div>
    <div class="avatar-back">
      <img
        src={backSrc}
        alt={`${alt} (back)`}
        class="avatar-img"
        width={size}
        height={size}
        loading="lazy"
      />
    </div>
  </div>
</div>

<style>
  .avatar-flip-container {
    perspective: 1000px;
    cursor: pointer;
    flex-shrink: 0;
    display: inline-block;
  }

  .avatar-flipper {
    position: relative;
    width: 100%;
    height: 100%;
    transition: transform 0.6s cubic-bezier(0.4, 0, 0.2, 1);
    transform-style: preserve-3d;
  }

  .avatar-flip-container.flipped .avatar-flipper {
    transform: rotateY(180deg);
  }

  .avatar-front,
  .avatar-back {
    position: absolute;
    inset: 0;
    backface-visibility: hidden;
    border-radius: 50%;
    overflow: hidden;
    box-shadow: 0 2px 12px rgba(0, 0, 0, 0.12);
  }

    transform: rotateY(180deg);
  }

  .avatar-img {
    width: 100%;
    height: 100%;
    object-fit: cover;

    border-radius: 50%;
  }

<script>
  const containers = document.querySelectorAll('.avatar-flip-container');
  containers.forEach(container => {
    container.addEventListener('click', () => {
      container.classList.toggle('flipped');
    });
```

```astro
---
import { getCollection } from 'astro:content';
import i18next from '@/i18n';
import Search from '@/components/search';
import PageLayout from '@/components/layouts/PageLayout.astro';
import slateConfig from '~@/slate.config';

const postCollection = await getCollection('post', ({ data }) => {
  return import.meta.env.DEV || data.draft !== true;
});

const tagCounts = postCollection.reduce<Record<string, number>>(
  (res, post) => {
    const postTags = post.data.tags;
    if (!postTags || !postTags.length) return res;
    postTags.forEach((tag) => {
      if (tag.trim() === '') return;
      if (res[tag]) { res[tag]++; } else { res[tag] = 1; }
    });
    return res;
  },
  { [i18next.t('common.all')]: postCollection.length },
);

const tags = Object.keys(tagCounts).map((tag) => ({ name: tag, count: tagCounts[tag] }));

const posts = [...postCollection]
  .sort((a, b) => {
    const dateA = (a.data.pubDate || a.data.published || new Date(0)).getTime();
    const dateB = (b.data.pubDate || b.data.published || new Date(0)).getTime();
    return dateB - dateA;
  })
  .map((post) => ({
    id: post.id,
    slug: post.slug,
    url: `/blog/${post.data.permalink || post.slug}`,
```
